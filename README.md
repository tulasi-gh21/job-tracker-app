# Job Application Tracker

## 📌 Description
A Job Application Tracker web application built using HTML, CSS, and JavaScript.  
It allows users to add, display, and delete job entries such as company name, role, and status. The data is stored in the browser using localStorage, so it remains saved even after refreshing the page.

---

## ✨ Features
- Add job application details (Company, Role, Status)  
- Display job entries in a list  
- Delete job entries  
- Data persistence using localStorage (data stays after refresh)  
- Simple and clean user interface  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript (DOM Manipulation)  
- localStorage (Browser Storage API)  

---

## 🎯 Purpose of the Project
This project was created to practice frontend development skills, especially working with JavaScript DOM manipulation, event handling, and browser storage (localStorage). It helps simulate a basic job tracking system.

## 🚀 How to Run the Project
1. Download or clone the repository  
2. Open the project folder  
3. Open index.html in any web browser  
4. Start adding job applications
